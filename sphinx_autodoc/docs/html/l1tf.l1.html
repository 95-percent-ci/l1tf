1. L1 Trend Filtering¶

This is implementation of L-1 Trend Filtering as published in the paper S.-J. Kim, et al.,L-1 Trend Filtering,Society for Industrial and Applied Mathematics (2009)

CVXOPT ( Python Software For Convex Optimization ) is used directly for solving quadratic programming. User is requested to go through its documentation once. Link is provided below.

l1tf.l1.l1_trend(signal, lamda, show_progress, iter_max=1000, tol_abs=1e-07, tol_rel=1e-06, feas_rel=1e-07, refine=0)[source]¶

This function checks if L-1 trend filtering can be done or not on the signal and initiates l1 trend filtering algorithm

Parameters

  • signal (numpy.ndarray) – Input signal.( a time series data )

  • lamda (float) – Regularization parameter.( to control trade off between the smoothness and residual )

  • iter_max (int) – Maximum number of solver iterations

  • tol_abs (float) – Absolute tolerance ( Required difference between primal and dual objective function )

  • tol_rel (float) – Relative accuracy (step tolerance, iteration ends when solver step size is less than this value )

  • feas_rel (float) – Tolerance for feasibility conditions ( Tolerance on dual feasibility )

  • refine (int) – Number of iterative refinement steps when solving KKT equations, for Quadratic problem it is 0.

  • show_progress (Bool) – show if display of solver progress required or not.

Returns

trend_ext: Extracted trend from the signal

Return type

numpy.ndarray
