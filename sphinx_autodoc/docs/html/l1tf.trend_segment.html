1. Result Extraction¶

Functions defined here are used to extract all the linear segment once L1 trend is computed.

class l1tf.trend_segment.TrendResult(change_point_ind, trend, signal)[source]¶

Bases: object

This class instantiates an object will following attributes: slope, intercept, start, end, segment_trend,signal_ext, residual

Variables

  • slope (float) – slope of the linear segment

  • intercept – intercept of the linear segment.

  • start (int) – start index of the linear segment.

  • end (int) – last index of the linear segment

  • segment_trend (numpy.ndarray) – extracted trend.

  • signal_ext (numpy.ndarray) – values of signal for the segment.

  • residual (float) – residue of fit for the segment

l1tf.trend_segment.get_result(trend, signal, change_points)[source]¶

This function returns fit results for each segment. Values for each segment are stored in a list.

Parameters

  • trend (numpy.ndarray) – L1 trend of signal

  • signal (numpy.ndarray) – Signal

  • change_points (numpy.ndarray) – change points in the extracted trend

Returns

A tuple of dictionary,where

  • in the first element

    • key is ‘segment_result’

    • values are stored in a list. Each list element is of l1tf.trend_segment.TrendResult datatype.

  • in the second element

    • key is ‘overall_mse’

    • stores total mean square error between the trend and the signal.

Return type

dict
